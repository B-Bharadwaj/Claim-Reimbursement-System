# Claim Reimbursement System

A work-in-progress web application built using **Django** and **OCR microservices** to simplify claim submission and document verification. The goal is to create a modular system where clients can upload required documents and analysts can validate claims efficiently.

---

## Features (WIP)
- User registration & login  
- Submit and track claims  
- Upload ID, PayG, and collateral documents  
- Independent OCR microservices for data extraction  
- Basic backend API structure implemented  
- Analyst workflow (planned)

---

## Tech Stack
- **Backend:** Django / DRF  
- **OCR Services:** Python (modular microservices)  
- **Database:** PostgreSQL  
- **Containerization:** Docker (planned)
